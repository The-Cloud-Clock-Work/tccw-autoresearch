description: Git rules for autoresearch experiments
globs: "**/*"
alwaysApply: true
---

- Commit messages: imperative mood, concise, prefix with area (e.g. "test: add edge case for X")
- One logical change per commit
- Check `git status` before committing
- Never force-push, never push to remote
- Never merge or rebase — the engine handles branch management
